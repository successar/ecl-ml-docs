CO_EFF
Tree
Generate time series data for demonstration of TS attributes. Generates four series of values. A lag period is inferred by the number of start values. The series generated are: flat (a random walk); a slope constructed by adding the additive increment product of the prior value and the multiplicative increment; and the cummulative values of the slope value.

Parameters
test records
start values, seasonality or cyclic behavior is entered with this parameter
noise mean, the mean value of normally distributed noise which will be added
noise variance, the variance of the normally distributed noise
multiply value, the multiplicative increment
additive value, the additive increment
Returns
the generated time series.
