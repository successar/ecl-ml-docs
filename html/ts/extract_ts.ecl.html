MACRO : extract_ts(dsIn, fieldName)
Inherit Type : local
@content : Extract the univariate time series. The data is required to be in time order, and the time perioods are reqiured to be regular. Missing values are not supported. The user must prepare the file such that these assumptions are met.
@return : a Univariate time series dataset
@param : dsIn the input time series source data set
@param : fieldName the input field name
